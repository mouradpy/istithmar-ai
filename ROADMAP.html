08 — Build Roadmap (6 Months · Full Time)
01
Foundation & Data Collection
Weeks 1-4
  • Set up project structure: Next.js + Node.js + TypeScript monorepo
  • Initialize Supabase project with PostgreSQL + pgvector
  • Build data collection pipeline: Python Scrapy for Moroccan sites
  • Collect initial dataset: 1000+ restaurants, hotels, businesses in Casablanca
  • Set up Google Places API integration for reviews data
  • Create database schema: users, businesses, analyses, reports
✅ OUTPUT: Database with real Moroccan business data · Scraping pipeline ready
02
Core Backend APIs
Weeks 5-7
  • Set up Node.js + Express server with TypeScript
  • Implement tRPC routers for type-safe APIs
  • Add Better Auth with JWT + email/password + Google OAuth
  • Create CRUD endpoints: businesses, analyses, user profiles
  • Implement Zod validation for all inputs
  • Add Redis caching for frequent queries (hotels, restaurants)
  • Set up Meilisearch for full-text search
✅ OUTPUT: Fully functional REST/TRPC APIs · Authentication · Search
03
Web Frontend MVP
Weeks 8-10
  • Initialize Next.js 15 with App Router and TypeScript
  • Set up Tailwind CSS + shadcn/ui components
  • Build RTL layout for Arabic support
  • Create main input form: budget, city, business idea
  • Connect to tRPC APIs via TanStack Query
  • Build results dashboard with gauge charts
  • Add Mapbox integration for competitor mapping
  • Implement user authentication pages
✅ OUTPUT: Working web app — users can input ideas and see results
04
AI Integration (Phase 1)
Weeks 11-13
  • Set up Vercel AI SDK with OpenAI GPT-4o
  • Implement LangChain for RAG pipelines
  • Create embeddings pipeline with Cohere + pgvector
  • Build success prediction prompt: analyze budget + idea + location
  • Implement competitor analysis using vector similarity
  • Add alternative idea generation when score is low
  • Set up LangFuse for monitoring AI calls
✅ OUTPUT: AI can analyze ideas and generate alternatives with 60%+ accuracy
05
Advanced Features
Weeks 14-16
  • Build interactive heat maps with competitor density
  • Implement live data ticker with real-time updates
  • Create business model canvas generator (AI-powered)
  • Add PDF report generation with React PDF Renderer
  • Build expense calculator for business planning
  • Implement background jobs for report generation (BullMQ)
  • Add email notifications for report ready
✅ OUTPUT: Full-featured platform with maps, canvas, PDF reports
06
Mobile App (React Native)
Weeks 17-20
  • Initialize Expo app with TypeScript
  • Set up navigation (React Navigation) with RTL support
  • Reuse web components where possible (shared UI logic)
  • Build main input screen with form validation
  • Implement results dashboard with charts
  • Add maps integration (react-native-maps)
  • Set up biometric authentication (FaceID/Fingerprint)
  • Implement offline support with local storage
✅ OUTPUT: iOS and Android apps with core functionality
07
Darija Fine-tuning & Production Polish
Weeks 21-24
  • Collect Moroccan Darija dataset for business contexts
  • Fine-tune model using Axolotl + LoRA
  • Implement language detection (Darija/Arabic/French)
  • Set up A/B testing for AI prompts
  • Comprehensive testing with Jest + Playwright
  • Set up Sentry + LogRocket monitoring
  • Dockerize entire stack for production
  • Deploy to Vercel + Cloudflare Workers
🚀 PRODUCTION READY: Full platform with Darija support, monitoring, deployment
